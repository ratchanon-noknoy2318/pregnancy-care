# KPPMCH Pregnancy Care
**Maternal Healthcare Automation System**

---

### Technical & Architecture
| Category | Detail | Implementation |
| :--- | :--- | :--- |
| **Stack** | Next.js 14, LINE API | App Router & Serverless |
| **Data** | GAS Webhook, Sheets | Asynchronous Operations |
| **Logic** | Clean Architecture | Decoupled Business Logic |
| **Deploy** | Vercel | Automated CI/CD |

---

### Configuration (`.env`)
| Variable | Description |
| :--- | :--- |
| `LINE_ACCESS_TOKEN` | Messaging API token |
| `LINE_SECRET` | Channel secret validation |
| `GAS_WEBHOOK_URL` | Google Apps Script endpoint |

---

### Quick Start & Contact
- **Setup:** `npm install && cp .env.example .env.local`
- **Run:** `npm run dev`
